# CropCast - AI-Powered Sustainable Agriculture

Leverage AI to predict crop yields and receive personalized recommendations for sustainable farming.

## Features
- **Personalized Recommendations**: Input farm data (pH, Nitrogen, Rainfall, etc.) to get AI-driven advice on irrigation, fertilization, and planting.
- **Yield Prediction**: Upload agricultural data in CSV format to predict potential crop yields and get actionable insights.
- **Modern UI**: A smooth, particle-animated background with a clean, light interface.

## How to push to GitHub

To push this code to your repository (`https://github.com/KaleSourav/Crop-Yield-Prediction.git`), follow these steps in your local terminal:

1. **Initialize Git** (if not already initialized):
   ```bash
   git init
   ```

2. **Add the Remote Repository**:
   ```bash
   git remote add origin https://github.com/KaleSourav/Crop-Yield-Prediction.git
   ```

3. **Stage and Commit your changes**:
   ```bash
   git add .
   git commit -m "Initial commit: AI-powered Crop Yield Prediction and Recommendation system"
   ```

4. **Set branch to main**:
   ```bash
   git branch -M main
   ```

5. **Push to GitHub**:
   ```bash
   git push -u origin main
   ```

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **AI**: Genkit with Google Gemini 1.5 Flash
- **Styling**: Tailwind CSS, ShadCN UI
- **Animations**: OGL (WebGL) for particles
