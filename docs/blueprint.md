# **App Name**: CropCast

## Core Features:

- Yield Prediction: Predict crop yields using historical weather and soil data as a tool, offering farmers data-driven insights for better planning.
- Intuitive Visualization: Display yield predictions in an intuitive format. Focus on a clean UI with clear visualizations such as graphs, maps, and charts showing predicted yields, and soil conditions.
- Custom Data Input: Allow users to input specific soil and weather data for custom yield predictions based on location.
- Actionable Insights: Display recommendations for optimized planting and harvesting times.

## Style Guidelines:

- Primary color: Earthy Green (#8FBC8F) to represent growth and nature.
- Background color: Light Beige (#F5F5DC), to evoke a natural and grounded feeling.
- Accent color: Warm Gold (#D4AF37), suggesting fertile soil, wealth, and prosperity.
- Body and headline font: 'PT Sans', a humanist sans-serif, to combine a modern look with warmth and personality. This provides a friendly, easily approachable user experience that aligns well with the application's goal of connecting farmers with technology.
- Use minimalist icons to represent different data points.
- Employ a grid-based layout for a balanced, easily understandable presentation of data and insights. Prioritize a design that adapts to different screen sizes, ensuring usability across devices used in the field.
